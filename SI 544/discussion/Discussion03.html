Part 2: Normal distribution, with mean and standard deviation

In R, we can use function rnorm to obtain random samples from a certain Normal distribution. For this part, do not worry about understanding the code. Focus on the concepts first, we will get to the code later in the semester.

Step 1: simulate a few random variables

In df, we create four columns with different mean and standard deviation. Let Normal(0,1) denote the standard normal distribution, with mean 0 and standard deviation 1. In the following chunk, we simulate the distribution of four normal random variables, where: * v01 ~ Normal(0,1) * v02 ~ Normal(0,2) * v12 ~ Normal(2,1)

library(ggplot2)
set.seed(1)
the_mean <- 0
the_standard_deviation <- 1
sim_count = 1000000 # a million simulations
df <- data.frame(v01 = rnorm(sim_count,
                            mean = the_mean, 
                            sd = the_standard_deviation
                 ), 
                 v02 = rnorm(sim_count,
                            mean = the_mean,
                            sd = the_standard_deviation * 2
                             ),
                 v21 = rnorm(sim_count,
                            mean = the_mean + 2,
                            sd = the_standard_deviation
                 ),
                 v05 = rnorm(sim_count,
                            mean = the_mean,
                            sd = the_standard_deviation * 5
                             )
                 )

In the simulation above, we repeatedly generated 10^{6} realizations from the Normal distribution, with mean being 0 and standard deviation being 1.

Take a look at df.

Density of Normal distribution

Mathematically, we have an exact representation of the Normal distribution. In R, function dnorm represents such bell-shaped curve.

ggplot(df, aes(x = v01)) + 
geom_histogram(aes(y =..density..),
               colour = "black",
               bins = 100,
               fill = "white") +
    stat_function(fun = dnorm, args = list(mean = mean(df$v01), sd = sd(df$v01)), color="red") +
    geom_vline(xintercept = sd(df$v01), color='blue') +
    geom_vline(xintercept = - sd(df$v01), color='blue') +
    geom_vline(xintercept = sd(df$v01) * 2, color='red') +
    geom_vline(xintercept = - sd(df$v01) * 2, color='red')

Calculate the Median of total confirmed cases

Notes on using summarise() function: 1. Dont worry about the following warning message:

`summarise()` ungrouping output (override with `.groups` argument)
  1. That summarize() function generates a new dataframe, and
  2. That when using summarize(), always give names to the new columns in the new dataframe

State_median = US_statelevel_bySunday %>%
